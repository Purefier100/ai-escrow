# 🚀 ChainEscrow — AI-Powered Freelance Escrow (GenLayer)

## 🧠 Overview

ChainEscrow is a decentralized freelance marketplace powered by **GenLayer AI smart contracts**.

It allows clients to post jobs and freelancers to submit work. Instead of relying on trust or intermediaries, **AI verifies whether the submitted work meets the job requirements before releasing payment**.

> 💡 No middleman. No disputes. Just verified results.

---

## ✨ Features

* 🧾 **On-chain Job Board**
  Clients can post jobs directly on-chain.

* 👨‍💻 **Multiple Freelancer Submissions**
  Anyone can submit work via a URL.

* 🏆 **Client Winner Selection**
  Clients can review all submissions and select the best one.

* 🤖 **AI-Powered Verification (GenLayer)**
  Smart contracts use AI to verify if submitted work fulfills the job description.

* 💰 **Trustless Payment Release**
  Payment is only released after successful AI verification.

* 🔒 **Decentralized Escrow System**
  No need for third parties or platforms.

---

## 🏗️ How It Works

1. **Client posts a job**

   * A job is created and linked to an escrow contract.

2. **Freelancers submit work**

   * Submissions are made using a URL.

3. **Client selects a winner**

   * The best submission is chosen.

4. **AI verifies the work**

   * GenLayer AI checks if the submission meets the job requirements.

5. **Payment is released**

   * If verification passes, funds are released to the freelancer.
